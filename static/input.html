Latin scansion engine input format

The input to the web app is assumed to follow the style used by Clyde Pharr's well-known Aeneid reader. To wit:

  1. Capital letters, periods, question marks, commas, parentheses, and square brackets are permitted but ignored.
  2. The glides must be indicated by v and j, respectively. (Note that Pharr does not use a geminate for intervocalic j.)
  3. (Unicode) macrons must be used to indicate long monophthongs.
It is our intention to eventually support other input formats (i.e., without explicit glides, without macrons) if possible.
